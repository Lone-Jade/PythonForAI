# Python 人工智能程序设计期中实验-试题2

## 项目简介

本项目为 Python 期中实验作业，针对**中国34个省会/直辖市/特别行政区城市的旅行商问题（TSP）**，分别实现了：
- 基于**模拟退火算法**的全局最优路径求解
- 基于**2-opt局部优化算法**的快速路径优化（仅用于对比分析）
- 算法性能对比与可视化分析

---

## 项目结构说明

```plaintext
├── CityTravel_main.py # 主程序：模拟退火算法实现
├── CityTravel_2-opt.py # 2-opt局部优化算法实现
├── settings.py # 全局参数配置（城市坐标、超参数）
├── data.csv # 城市经纬度数据文件
├── figures # 运行结果图片
	├── tsp_best_route_*.png # 最优路径可视化结果
	├── tsp_distance_curve_*.png # 算法收敛过程曲线
├── 实验报告.pdf/.docx # 期中实验报告文档
└── README.md # 项目说明文件（本文档）
```

---

## 各文件功能说明

### 1. 核心代码文件
- `CityTravel_main.py`
  - 实现基于**模拟退火算法**的TSP求解
  - 包含贪心初始化、Metropolis准则接受机制、邻域路径扰动
  - 输出最优路径、最短距离及收敛过程曲线
  - 支持超参数调优对比实验

- `CityTravel_2-opt.py`
  - 实现**2-opt局部优化算法**求解TSP
  - 用于和**模拟退火算法**对比分析

- `settings.py`
  - 定义全局超参数：初始温度、降温速率、迭代次数等
  - 统一管理配置，便于对比实验与参数调优

- `data.csv`
  - 存储34个中国城市的经纬度坐标数据
  - 为算法提供计算城市间欧氏距离的数据源

### 2. 实验结果文件
- `tsp_best_route_distance_*.png`：不同算法/参数下的最优路径可视化结果
- `tsp_distance_curve_distance_*.png`：模拟退火算法的收敛过程曲线（距离随迭代变化）

### 3. 实验报告
- `实验报告.pdf/.docx`：期中实验报告文档，为防止格式错乱，除按照要求提交Word版实验报告外，也提交PDF版实验报告

## 运行说明
1. 环境依赖：
```python
Python 3.x
numpy 2.4.2+
matplotlib 3.10.8+
```
2. 运行方式：

   - 主程序：`python CityTravel_main.py`

   - 2-opt算法：`python CityTravel_2-opt.py`

3. 输出：控制台打印最优路径与距离，同时自动保存可视化结果到当前目录

---

## 实验说明
- 算法对比：对比了模拟退火（含不同超参数配置）与2-opt算法的求解效果
- 核心结论：模拟退火全局搜索能力强，可避免局部最优；2-opt运行速度快，适合小规模数据场景；二者结合可兼顾精度与效率
- 详细实验过程、结果分析与结论请参考《实验报告.word/.pdf》
